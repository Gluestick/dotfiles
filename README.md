# Dotfiles
This is where I keep all my configuration files for my GNU/Linux environments.
This repository contains dotfiles for the following applications.
* git
* vim
* Zsh
* Urxvt
* i3
    * dmenu-xft
    * pcmanfm
    * mpc
* conky
* vimperator
* luakit

### Fonts
You will need the following fonts installed on your system.
* Inconsolata
